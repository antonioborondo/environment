# Install toolchain

Open ConEmu and run the following command:
```
pacman -S mingw-w64-x86_64-toolchain mingw-w64-x86_64-cmake
```

## Add toolchain to the PATH

Add the following folder to the PATH: `C:\msys2\mingw64\bin`
